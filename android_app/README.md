# 睡眠监护 Android App

这是监护人手机端 App。当前版本采用后端轮询模式：

- 后端轮询 `http://192.168.31.236:8081`，远程接收心率、呼吸率、目标距离、温湿度、呼噜分数和分贝、SOS、疑似呼吸暂停等报警。
- 雷达板通过 WiFi+UDP 直连后端，App 不再直连雷达板蓝牙。

## 使用方式

1. 用 Android Studio 打开 `android_app/`。
2. 连接安卓手机，编译并安装 App。
3. 打开 App，确认后端地址为 `http://192.168.31.236:8081`，点击“开始监听”。
4. 实时卡片显示后端汇聚的心率、呼吸率、目标距离、温湿度、打鼾情况。
5. 触发“小智救命”或疑似呼吸暂停后，手机会通过本地通知报警。

## 注意

- Android 13 及以上需要允许通知权限，否则报警通知无法弹出。
- 手机不要填写 `localhost` 作为后端地址，应使用电脑局域网 IP。
- 已安装的旧版本 App 可能缓存了旧的后端地址，需要在 App 内重新填写 `http://192.168.31.236:8081` 或清除应用数据。
