# 睡眠监护 Android App

这是监护人手机端 App。当前版本采用“双模式”：

- 蓝牙直连雷达板 `RadarCare-E84`，近场查看心率、呼吸率、目标距离、板子静止状态。
- 后端轮询 `http://192.168.0.102:8081`，远程接收 SOS、疑似呼吸暂停、温湿度、呼噜分数和分贝。

## 使用方式

1. 用 Android Studio 打开 `android_app/`。
2. 连接安卓手机，编译并安装 App。
3. 打开 App，确认后端地址为 `http://192.168.0.102:8081`，点击“开始监听”。
4. 点击“扫描雷达”，允许蓝牙权限后等待发现 `RadarCare-E84`。
5. 点击“连接”，实时卡片会显示蓝牙雷达数据。
6. 触发“小智救命”或疑似呼吸暂停后，手机会通过本地通知报警。

## 注意

- 蓝牙直连只适合近场查看，远程报警仍建议保持后端监听开启。
- Android 12 及以上需要允许蓝牙扫描/连接权限。
- Android 13 及以上需要允许通知权限。
- 手机不要填写 `localhost` 作为后端地址，应使用电脑局域网 IP。
